Learning Path

Follow these steps in order:

  1. HTML — Learn the structure of web pages (headings, links, images, forms)
  2. CSS — Learn how to style pages (colors, fonts, layout)
  3. JavaScript — Add interactivity (buttons, animations, logic)
  4. Build Projects — Practice by making real websites

Best Free Websites to Study

  • MDN Web Docs — The most complete and accurate reference for HTML, CSS & JavaScript
  • freeCodeCamp — Free full curriculum with lessons and certificates
  • W3Schools — Short lessons with live examples you can try in the browser
  • The Odin Project — Free open-source path from beginner to full-stack
  • Codecademy — Interactive lessons, no setup needed
